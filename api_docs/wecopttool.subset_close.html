Check if the first set set_a is contained, to some tolerance, in the second set set_b.

Parameters:

  • set_a (FloatOrArray) ‚Äì First array which is tested for being subset.

  • set_b (FloatOrArray) ‚Äì Second array which is tested for containing set_a.

  • rtol (float) ‚Äì The relative tolerance parameter. Passed to numpy.isclose().

  • atol (float) ‚Äì The absolute tolerance parameter. Passed to numpy.isclose().

  • equal_nan (bool) ‚Äì Whether to compare NaNs as equal. Passed to numpy.isclose().
