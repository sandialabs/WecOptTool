wecopttool.waves

Provide the wave definition structure and provide functions for creating different common types of waves.

This module provides the (empty) dataset structure for waves in wecopttool. It also provides functions for creating common types of waves such as regular waves and irregular waves. The dataset structure is an xarray.Dataset containing the following two 2D xarray.DataArrray: (1) the amplitude amplitude (m^2) and (2) the phase phase (rad). The 2D coordinates are: wave angular frequency omega (rad/s) and direction wave_direction (rad).

Two omni-directional spectra and one spread function are included. These serve as examples for creating your own omni-directional spectra and spread functions, possibly leveraging other libraries (e.g., MHKiT).

Functions
