wecopttool.core.write_netcdf

write_netcdf(fpath, data)[source]

Save an xarray.Dataset with possibly complex entries as a NetCDF file.

Can handle complex entries in the NetCDF by using capytaine.io.xarray.separate_complex_values()

Parameters:

  • fpath (str | Path) – Name of file to save.

  • data (Dataset) – Dataset to save.

Return type:

None
